# INFS3830---Social-Media-Analytics
In the **INFS3830 Social Media Analytics course**, I worked on a group project analysing social media discussions related to **Volkswagen and Toyota**to understand how online conversations reflect public perception and brand reputation. The goal was to evaluate **how customers react to major corporate issue on social media** and how companies’ social media strategies influence these discussions.

The project used a combination of **text analytics** and**social network analysis** to analyse social media data from Twiiter. We used **SAS Visual Text Analytics** to perform text mining, keyword extraction, and topic analysis to identify the most frequently discussed themes associated with the two brands. Sentiment analysis conducted in **R Studio** was applied to classify posts into positive, negative, and neutral categories to compare overall public attitudes.

To further analyse user interactions, we conducted social network analysis using **NodeXL**. This allowed us to visualise discussion networks and identify influential users who played a key role in spreading or shaping conversations about the brands. Network metrics and visualisations helped reveal the structure of the discussions and the relationships between participants.

**Excel** was used for data cleaning and preparation as well as engagement analysis. We created data visualisations in Excel to examine discussion frequency, user interaction patterns and engagement trends across the dataset.

By combining text analytics, sentiment analysis, and social network analysis, the project transformed social media data into insights about brand reputation, public sentiment, and engagement patterns. The goal of this project is to **help businesses use social media analytics to design data-driven social media strategies**, including identify key influencers in online discussions, and improve how brands communicate with customers during major corporate events.
