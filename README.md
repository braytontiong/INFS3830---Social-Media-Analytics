# INFS3830---Social-Media-Analytics
In the **INFS3830 Social Media Analytics course**, I worked on a group project analysing social media discussions related to **Volkswagen and Toyota**to understand how online conversations reflect public perception and brand reputation. The goal was to evaluate **how customers react to major corporate issue on social media** and how companies’ social media strategies influence these discussions.

The project used a combination of **text analytics** and**social network analysis** to analyse social media data from Twiiter. We used **SAS Visual Text Analytics** to perform text mining, keyword extraction, and topic analysis to identify the most frequently discussed themes associated with the two brands. Sentiment analysis was applied to classify posts into positive, negative, and neutral categories to compare overall public attitudes.

To further analyse user interactions, we conducted social network analysis using **NodeXL**. This allowed us to visualise discussion networks and identify influential users who played a key role in spreading or shaping conversations about the brands. Network metrics and visualisations helped reveal the structure of the discussions and the relationships between participants.

Additional engagement analysis was conducted using **Excel** to examine patterns in discussion frequency and user interaction levels as well as providng data visualisation. By combining text analytics and network analysis, the project translated social media data into insights about brand reputation, public sentiment, and online engagement patterns, demonstrating how organisations can use social media analytics to **monitor public opinion and manage communication strategies.**
